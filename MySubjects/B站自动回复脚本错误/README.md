# B站自动回复工具

一个Windows端Python 3.12.3脚本，用于监控B站指定UID用户的新回复，并自动逐一回复指定自定义列表的内容。

## 功能特性

- 实时监控指定UID用户的新回复
- 自动逐一回复自定义列表内容（循环使用）
- 图形化界面(GUI)，操作简单
- 配置自动保存，下次启动自动加载
- 已回复记录持久化，避免重复回复
- 日志记录和实时显示
- 频率控制，避免触发B站风控

## 环境要求

- **Python 3.12.3** (推荐)
- Windows 7/10/11
- 依赖库: `requests`

## 安装步骤

### 1. 安装Python 3.12.3

从官网下载并安装: https://www.python.org/downloads/release/python-3123/

安装时请勾选 **"Add Python to PATH"**

### 2. 安装依赖

打开命令提示符(CMD)或PowerShell，执行：

```bash
pip install requests
```

### 3. 下载脚本

将 `bilibili_auto_reply.py` 保存到任意文件夹

## 使用方法

### 1. 启动程序

双击运行 `bilibili_auto_reply.py`，或在命令行中执行：

```bash
python bilibili_auto_reply.py
```

### 2. 配置Cookie

程序需要B站的Cookie才能正常工作，获取方法：

1. 使用Chrome/Edge浏览器登录B站 (https://www.bilibili.com)
2. 按 **F12** 打开开发者工具
3. 切换到 **"网络/Network"** 标签
4. 刷新页面(F5)
5. 在列表中找到任意一个 `api.bilibili.com` 的请求
6. 点击该请求，在右侧找到 **"请求头/Request Headers"**
7. 找到 **"Cookie"** 字段，复制其中的：
   - `SESSDATA=xxx` (分号前的部分)
   - `bili_jct=xxx` (分号前的部分)

将这两个值填入程序的对应输入框中。

### 3. 设置监控参数

- **目标用户UID**: 要监控的B站用户的UID（数字）
- **检查间隔**: 检查新回复的时间间隔（秒），建议30秒以上

### 4. 设置回复内容

在"回复内容列表"文本框中，每行输入一条回复内容，例如：

```
感谢回复~
收到你的消息啦！
谢谢支持！
欢迎常来交流~
```

程序会按顺序循环使用这些回复内容。

### 5. 开始监控

点击 **"开始监控"** 按钮，程序将开始：
- 定期检查目标用户的新回复
- 自动回复预设的内容
- 在日志区域显示运行状态

### 6. 停止监控

点击 **"停止监控"** 按钮即可停止。

## 配置文件

程序会自动生成以下配置文件：

- `config.json` - 保存用户设置（Cookie、UID、回复内容等）
- `replied.json` - 记录已回复的评论ID，避免重复回复
- `bilibili_auto_reply.log` - 运行日志文件

## API说明

本脚本使用的B站API：

| 功能 | API地址 | 方法 |
|------|---------|------|
| 获取回复消息 | `https://api.bilibili.com/x/msgfeed/reply` | GET |
| 发送评论回复 | `https://api.bilibili.com/x/v2/reply/add` | POST |
| 获取用户信息 | `https://api.bilibili.com/x/space/acc/info` | GET |

## 注意事项

1. **Cookie安全**: 
   - Cookie包含登录凭证，请勿泄露给他人
   - 建议定期更换密码
   - Cookie有效期约1-2个月，过期后需要重新获取

2. **频率限制**:
   - 检查间隔建议设置为30秒以上
   - 每次回复后有5秒延迟
   - 如触发风控，程序会自动等待60秒

3. **使用限制**:
   - 仅回复目标UID用户的评论
   - 每个评论只回复一次（已回复记录会保存）
   - 需要保持程序运行才能持续监控

## 常见问题

### Q: 提示"Cookie可能已过期"怎么办？
A: 重新获取Cookie并填入程序。

### Q: 如何获取用户的UID？
A: 访问用户主页，URL中的数字就是UID，例如：`https://space.bilibili.com/12345678`，UID就是12345678。

### Q: 可以监控多个用户吗？
A: 当前版本只支持监控单个用户，如需监控多个用户，可以运行多个程序实例。

### Q: 回复内容支持表情吗？
A: 支持B站表情代码，例如 `[微笑]`、`[doge]` 等。

### Q: 程序可以最小化到托盘吗？
A: 当前版本不支持，需要保持窗口运行。可以最小化窗口到任务栏。
